# LewisAPIs
This is a list of 40 APIs every developer must know from the [video by coding with lewis](https://www.youtube.com/watch?v=YHxj3LvZoLQ)

Feel free to contribute :)

1. **Unsplash API**

- **Description:** Provides free, high-resolution images and videos.
- **URL:** [https://unsplash.com/developers](https://unsplash.com/developers)
